🌤️ Weather Checker
A sleek, real-time weather application designed to help users check atmospheric conditions before heading out. This project features a clean interface with detailed environmental data, including air quality and visibility metrics.

Live Demo: https://cheakweatherbysaiful-73.netlify.app/

🚀 Features
Real-Time Search: Get instant weather updates for any city worldwide.

Detailed Atmospheric Data: - Temperature & "Feels Like" metrics.

Humidity, Wind Speed, and Pressure levels.

UV Index and Visibility (km).

Air Quality Integration: Provides health-critical data including PM2.5, PM10, and the US EPA Index.

Dynamic UI: Clean, centered card layout with responsive design for mobile and desktop viewing.

API Integration: Powered by WeatherAPI for accurate and up-to-date global forecasts.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)

Styling: Utility-first CSS for a modern, rounded-corner UI.

API:  for fetching live meteorological data.

Deployment: Netlify

📂 Project Structure
⚙️ How It Works
The user enters a city name (e.g., "London" or "Chattogram") into the search bar.

The app sends an asynchronous request to the WeatherAPI.

The UI updates dynamically to display temperature, wind conditions, and a specific "Air Quality" card for localized health insights.
